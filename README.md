# LingoLearn

Welcome to **LingoLearn**! Your go-to platform for mastering new languages through interactive learning. We believe in making language education engaging and accessible for all.

## Features
- **Interactive Lessons**: Dive into lessons that adapt to your skill level.
- **Video Tutorials**: Access our library of video tutorials.
- **Community Support**: Join our community to connect, practice and grow together.

## Installation
1. Clone the repository: `git clone https://github.com/Prateek1771/LingoLearn.git`
2. Navigate to the directory: `cd LingoLearn`
3. Run the app: `npm start`
